Literal

public class Literal<T>

Literals are used by DataTypes to be able to recognise static values, that can be expressed as a given type

  • init(convert:)

    In case of more complicated expression, this initialiser accepts a convert block, which can be used to process static values. Return nil, if the input cannot be accepted and converted.

  • init(_:convertsTo:)

    In case the literals are easily expressed, static keywords, then this initialiser is the best to use. The first parameter is the used keyword; the second one is the statically typed associated value. As it is expressed as an autoclosure, the provided expression will be evaluated at recognition time, not initialisation time. For example, Date() is perfectly acceptable to use here.
